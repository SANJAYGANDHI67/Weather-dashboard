# 🌦️ Weather Dashboard

A modern Weather Dashboard web application built using HTML, CSS, JavaScript, OpenWeather API, and Local Storage.

The application helps users check real-time weather conditions, forecast future weather, track recent searches, and view weather information with an interactive and responsive user interface.

## 🚀 Live Demo

https://sanjaygandhi67.github.io/Weather-dashboard/

---

## 📌 Features

### 1. Current Weather Information

* Search weather by city name
* Real-time weather updates
* Current temperature
* Weather condition
* Weather icon
* Humidity information
* Wind speed information

### 2. Location-Based Weather

* Detect current user location
* Fetch weather using Geolocation API
* One-click weather access

### 3. Search Suggestions

* City autocomplete suggestions
* Fast city search experience
* Improved user interaction

### 4. 5-Day Weather Forecast

View upcoming weather forecasts including:

* Day name
* Weather icon
* Temperature
* Weather condition

### 5. Date & Time Information

Displays:

* Current date
* Current time
* Sunrise time
* Sunset time

### 6. Dynamic Weather Backgrounds

Background changes automatically based on weather conditions:

* ☀️ Sunny → Orange
* ☁️ Cloudy → Gray
* 🌧️ Rainy → Blue
* ⛈️ Storm → Dark
* 🌙 Default → Dark Blue

### 7. Search History

* Stores recent searched cities
* Quick access to previous searches
* Local Storage integration

### 8. Theme Toggle

* Dark Mode
* Light Mode
* Smooth user experience

### 9. Loading & Error Handling

* Loading spinner while fetching data
* Invalid city error messages
* Better user feedback

### 10. Responsive Design

* Desktop friendly
* Tablet friendly
* Mobile friendly
